# C Console Mini Games

This is a C console application that includes three interactive mini-games:

1. **Guess the Number** – Try to guess the secret number.
2. **Rock Paper Scissors** – Classic game against the computer.
3. **Math Challenge** – Solve random addition and subtraction questions.

## Features
- Written in **C programming language**.
- Uses **functions, loops, and conditional statements**.
- Includes **console color styling** for better user experience.
- Interactive menu and scoring system.

## Skills Demonstrated
- C programming fundamentals
- Problem-solving and logical thinking
- User interface design in console
- Random number generation

## How to Run
1. Download or clone this repository.
2. Compile the `main.c` file using a C compiler (like GCC or Dev-C++).
3. Run the executable and enjoy the mini-games!
